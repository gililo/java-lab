# Exercise 01 - Java Control Structures

## Description

Write a program that receives a number from the user and generates its multiplication table from 1 to 10.

```text
Number x 1 = result
Number x 2 = result
...
Number x 10 = result
```

## Example

### Input

```text
Type a number: 5
```

### Output

```text
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
```

## Code Overview

The project contains two classes:

- `Main`: responsible for receiving the user's input and calling the multiplication table generator.
- `MultiplicationTableGenerator`: responsible for generating and displaying the multiplication table from 1 to 10.
