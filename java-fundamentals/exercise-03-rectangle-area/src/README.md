# Exercise 03 - Fundamentals of Java Programming

## Description

Write a program that receives the base length and height of a rectangle, calculates its area, and displays the result on the screen.

```text
The area of the rectangle is: X
```

## Formula

```text
area = base × height
```

## Example

### Input

```text
Type the base length of the rectangle: 10
Type the height of the rectangle: 5
```

### Output

```text
The area of the rectangle is: 50.0
```
