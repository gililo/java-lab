# Exercise 04 - Fundamentals of Java Programming

## Description

Write a program that receives the name and age of two people, calculates the age difference between them, and displays the result on the screen.

```text
The age difference between Name 1 and Name 2 is X years.
```

## Example

### Input

```text
Type the name of the first person: John
Type the age of the first person: 25
Type the name of the second person: Mary
Type the age of the second person: 20
```

### Output

```text
The age difference between John and Mary is 5 years.
```
