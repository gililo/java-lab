# Exercise 01 - Fundamentals of Java Programming

## Description

Write a program that receives a person's name and birth year, then prints the following message on the screen:
```text
Hello, 'Name'. You are 'X' years old.
```
## Example

### Input
```text
Type a name: John
Type the birth year: 2000
```
### Output
```text
John, you are 26 years old.
```

## What I Practiced
- Creating classes in Java
- Reading user input with `Scanner`
- Working with variables
- Creating and calling methods
- Using `LocalDate` to get the current year
- Basic input validation

## Code Overview
The project contains two classes:
- `Main`: responsible for receiving the user's input and displaying the result.
- `AgeCalculator`: responsible for calculating the user's age based on the birth year.
