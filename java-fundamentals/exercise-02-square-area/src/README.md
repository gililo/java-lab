# Exercise 02 - Fundamentals of Java Programming

## Description

Write a program that receives the side length of a square, calculates its area, and displays the result on the screen.

```text
The area of the square is: X
```

## Example

### Input

```text
Type the side length of the square: 5
```

### Output

```text
The area of the square is: 25.0
```

## Project Structure

```text
exercise-02-square-area/
├── src/
│   ├── Main.java
│   └── SquareAreaCalculator.java
└── README.md
```

## Technologies Used

- Java
- Scanner

## What I Practiced

- Creating classes in Java
- Reading user input with `Scanner`
- Working with variables
- Creating and calling methods
- Performing mathematical calculations
- Organizing code using separate classes

## Code Overview

The project contains two classes:

- `Main`: responsible for receiving the user's input and displaying the result.
- `SquareAreaCalculator`: responsible for calculating the area of the square based on the side length.
